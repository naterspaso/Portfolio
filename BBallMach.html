DIY KrazyKart

by Nathan Su

This project is a combination of hardware and software aimed to interface and control various streaming-based music/video apps such as Spotify, Apple Music, YouTube, and YouTube Music. The goal is to design and test the system for better refinement and gain experience in both hardware and software development.

Project Specifications

  • User Identification: The controller defines a specific user to prevent music interruption by others.
  • Use Limitations: Not suitable for collaborative environments like study rooms.
  • Functionality: Only supports play/pause actions.
  • Environmental Compatibility: Best for indoor, dry environments. Not suitable for outdoor or wet areas.

Future Improvements

  • Enhanced User Interface: Implement a more intuitive and visually appealing design, and ensure responsiveness.
  • Voice Control Integration: Add voice command support, with multilingual capabilities for broader accessibility.
